## Norme de Programmation

Les différentes conventions de nommage qui existe en programmation

Le camelCase:
myUrl

Le PascalCase:
MyUrl

Le WikiCase:
MyUrl

Le Kebab Case:
my-url

Snake Case ou Underscore Case:
my_url
